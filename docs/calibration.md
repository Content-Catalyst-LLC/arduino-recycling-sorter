# Calibration Guide

## Why Calibration Matters

This sorter relies on simple threshold-based sensing and mechanical actuation. Performance depends on:

- object size and shape
- item speed through the chute
- ultrasonic sensor placement
- metal sensor sensitivity
- servo angle alignment
- chute geometry

## Practical Calibration Notes

1. Measure the effective object-detection range of the ultrasonic sensor
2. Confirm the metal sensor triggers consistently for intended objects
3. Adjust the servo angles for clean diversion into the target bins
4. Test with multiple item types, shapes, and sizes
5. Observe repeated runs to identify jams or misroutes

## Notes

This prototype performs simple sensor-based routing, not advanced material recognition. It should be described as a prototype sorting system, not a full autonomous recycling intelligence platform.
